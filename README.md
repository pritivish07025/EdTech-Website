# 🎓 EdTech Website Clone

A modern and responsive **EdTech Website Clone** built using **React, HTML5, CSS3, JavaScript, and Bootstrap**. This project replicates the landing page of an online learning platform with a clean user interface, responsive layout, and reusable React components.

🔗 **Live Demo:** https://edtechwebsite-clone.netlify.app/

---

## 📖 About The Project

The EdTech Website Clone is designed to provide a professional and engaging user experience for an online education platform. It features a modern landing page that highlights courses, instructors, testimonials, statistics, and contact information.

The project focuses on building a responsive frontend using React and Bootstrap while following component-based architecture for better maintainability.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/pritivish07025/edtechwebsite-clone.git
```

### Navigate to Project

```bash
cd edtechwebsite-clone
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
